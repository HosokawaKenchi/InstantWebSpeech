# Instant Web Speech

## 利用者向けマニュアル（PWA対応版）

**公開URL：<https://hosokawakenchi.github.io/InstantWebSpeech/>**


***

# 1. アプリの概要

主な特徴：

*   Web Speech API による軽快なリアルタイム音声認識
*   Webブラウザ上でインストール不要で動作
*   Progressive Web App (PWA)によるオフラインアプリ化も可能
*   多言語認識（UIも言語依存フレーズに自動更新）
*   認識結果を自動保存（localStorage）
*   最小限の機能・UI

***

# 2. 推奨動作環境

*   **Google Chrome**（PC / Android）
*   **Microsoft Edge**
*   **iOS Safari**
*   ブラウザが **Web Speech API** と **Service Worker** に対応している必要があります  
    （対応していない場合は、アプリ側が「Speech Recognition is not supported.」と表示します）
*   Firefoxではブラウザアプリとして動作しますがPWAによるオフラインアプリ化ができません。


***

# 3. Progressive Web App（PWA）としてローカルにインストールする

このアプリはProgressive Web App (PWA)用の設定があり、ローカル環境にインストールして普通のアプリのように扱うことができます。

***

## 3.1 PWAでできること

*   ホーム画面やPCアプリ一覧にアイコンが追加
*   URLバーなしの「アプリ風」単独表示
*   オフライン起動
*   ブラウザのタブに埋もれず起動しやすい
*   起動速度向上

***

## 3.2 スマートフォンでのインストール

### ● Android（Chrome）

1.  <https://hosokawakenchi.github.io/InstantWebSpeech/> を開く
2.  画面下の「ホーム画面に追加」または Chrome メニュー → **ホーム画面に追加**
3.  アイコンが追加されアプリとして起動可能

### ● iPhone / iPad（Safari）

1.  URLにアクセス
2.  共有メニュー（□↑）→ **ホーム画面に追加**
3.  名前を確認し「追加」

***

## 3.3 PCでのインストール

### ● Chrome / Edge

1.  URLを開く
2.  アドレスバー右端の **インストールアイコン** をクリック
3.  「インストール」を選択

***

## 3.4 PWAの技術的仕組み（簡易説明）

*   ローカルアプリのように動作する
    *   PWAインストールすると、通常のブラウザウインドウへの描画と異なりあたかも独立したアプリであるかのような表示になります。
    *   アプリアイコンが設定されています。
*   オフライン動作する
    *   初回アクセス時に静的リソースをキャッシュ
    *   オフライン時にキャッシュから提供
    *   ページ読み込みを高速化

***

# 4. 画面構成

画面は以下の構成です。

### (1) 言語選択ドロップダウン

認識したい言語を選択。

### (2) 録音開始／停止ボタン

音声認識をトグル。色変化で状態が分かります。

### (3) コピー（📋）ボタン

認識テキストをクリップボードへ。

### (4) 認識テキスト表示エリア

リアルタイムで文字が表示されます。

### (5) 削除（🗑️）ボタン

テキストと localStorage を消去。


***

# 5. 基本の使い方

## 5.1 言語選択

画面上部のドロップダウンから認識言語を選択します。  
選択内容は localStorage に保存され、次回アクセス時に復元されます。

## 5.2 音声認識を開始

1.  「録音開始」ボタンを押す
2.  ブラウザのマイク許可を承認
3.  認識がリアルタイムに画面へ反映されます

録音中はボタンが赤くなり `stop` UI へ変わります。

## 5.3 停止

ボタンを押すと録音停止。青色の `start` UI に戻ります。

## 5.4 コピー

📋 ボタンでテキスト内容をコピーできます。成功時は表示が一時的に `copyDone` に切り替わります。

## 5.5 削除

🗑️ ボタン → 確認ダイアログ → テキスト削除  
localStorage も消去されます。

***

# 6. トラブルシューティング

*   **音声認識が動かない**  
    → マイク許可、ブラウザ対応状況を確認
*   **インストールが表示されない**  
    → キャッシュ削除後に再読み込み  
    → iOS は Safari 専用手順のみ
*   **テキストが消えた**  
    → リセット操作で localStorage が空になります
